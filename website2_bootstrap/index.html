Esta página muestra el pronóstico de parámetros convectivos, principalmente para la parte centro-sur de Chile, calculados a partir de salidas del modelo Global Forecast System (GFS) con una resolución horizontal de 0.25º grados. Estos pronósticos pueden ser usados como una herramienta de ayuda para monitorear el establecimiento de condiciones favorables para la ocurrencia de tormentas severas o tornados/trombas marinas en Chile. En esta página los usuarios podrán encontrar mapas con la distribución horizontal de algunos parámetros convectivos, diagramas skew-T para un grupo de ciudades del centro-Sur de Chile, así como series temporales de algunos parámetros convectivos para estas ciudades. Los productos de esta página se actualizan una vez al día. La página está disponible tanto en español como en inglés, permitiendo al usuario acceder a la información de este sitio en cualquiera de estos idiomas. Es importante destacar que los productos de esta página no deben tomarse como un pronóstico oficial, sino solo como una guía para el seguimiento de condiciones favorables para la ocurrencia de tormentas severas.
